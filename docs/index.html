Claude/Codex에서 시작

에이전트 앱에 repo 링크를 주고 보일러 플레이트 시작해줘라고 말합니다.

승인 후 설치

설치 방법만 안내하고 끝내지 않고, 승인하면 직접 설치를 시도합니다.

Fallback 명령

PowerShell/터미널 명령은 고급 사용자, QA, 장애 복구용으로만 사용합니다.

설치 순서 — 7단계 (의존성 순서대로)
1

prereqs 기반 설치

모든 것의 토대. Xcode Command Line Tools(컴파일러·git·헤더)와 Homebrew를 깔고, brew 환경을 ~/.zprofile에 영구 등록합니다.

Xcode CLTHomebrew
2

brew CLI 패키지 (Brewfile)

선언적 Brewfile 하나로 개발 도구를 일괄 설치. 이후 모든 단계가 여기서 깔린 도구에 의존합니다.

↳ 필요: prereqs (Homebrew)

ghripgrepfdfzfbatjqtreewget miseuvrustupbunstarship colimadockercomposebuildxmoleNerd Font
3

runtimes 런타임 설치

mise로 언어 런타임을, rustup으로 Rust를 설치. 프로젝트별 버전 전환이 가능해집니다.

↳ 필요: brew (mise, rustup)

node @ltspython @latestgo @latestrust stablerust-analyzer
4

shell 셸 환경

oh-my-zsh + 플러그인, starship 프롬프트, Nerd Font를 연결. ~/.zshrc는 마커 블록으로 주입해 재실행해도 중복되지 않습니다.

↳ 필요: brew (starship/fzf), runtimes (mise)

oh-my-zshgit · npm · node · macosautosuggestionssyntax-highlightingstarship.tomlcmux (Nerd Font)
5

docker 컨테이너

Docker Desktop 없이 Colima로 컨테이너 런타임 구성. docker compose/buildx 플러그인을 연결합니다.

↳ 필요: brew (colima, docker)

Colimadocker CLIcomposebuildx
6

git Git & GitHub

git 신원(GitHub noreply 이메일 자동 도출), HTTPS 자격증명 도우미, 합리적 기본값을 설정. 기존 값은 덮어쓰지 않습니다.

↳ 필요: brew (gh)

user.name/emailgh authcredential helperdefaults
7

agents AI 코딩 에이전트 — 최종 단계

체인의 정점. Claude Code와 Codex를 중심으로 AI 에이전트를 준비하고, Matt Pocock Skills는 필수 setup으로 안내합니다. LazyCodex, Hermes Agent, Superpowers Debug/Verify Pack 같은 add-on은 필요할 때만 명시적으로 선택합니다.

↳ 필요: runtimes (node/npm), brew (bun)

몇 번을 돌려도 안전. 설정 파일은 마커 블록으로 교체되어 중복·손상이 없습니다.

bash 3.2 호환

맥 기본 셸로 동작. 갓 산 맥에서 추가 설치 없이 바로 실행됩니다.

비파괴적

기존 파일·설정은 보존. 빈 값만 채우고, 사용자 것은 덮어쓰지 않습니다.

Business judgment route

설치기는 사업성 판단을 직접 하지 않습니다. business viability/commercial judgment 질문은 보이는 G-stack office-hours repo/link로 보내고, 없으면 그때 사용자에게 요청합니다.

권한 (sudo)
Homebrew 설치

맥 비밀번호 1회 — 공식 installer가 물음 (brew 없을 때만).

Xcode CLT

GUI "설치" 클릭 (CLT 없을 때만).

그 외 전부

user 공간, sudo 없음 — mise·rustup·bun·dotfiles 모두 홈 디렉터리. 스크립트는 sudo를 직접 부르지 않음.

이미 도구가 깔린 맥에서
비파괴적

brew·oh-my-zsh·codex는 있으면 스킵. git 신원은 비었을 때만 설정 (덮어쓰지 않음).

node 가림 주의

기존 node(.pkg/nvm/brew)는 안 지워지고 mise node가 PATH로 가립니다. 감지 시 경고 출력 — which -a node로 확인.

.zshrc 중복

손으로 편집한 줄이 있으면 키트 블록이 추가돼 중복 실행(무해). 한 곳으로 정리 권장.
